## Tech Stack

This portfolio is built using **React** and **Vite**, providing a fast and modern development experience. It also includes ESLint for code quality and proper formatting.

- **React** – Frontend library for building UI components  
- **Vite** – Next-generation frontend tooling for fast development  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **Framer Motion** – Animations and transitions for smooth interactions  
- **ESLint** – Code linting for maintaining best practices  

## Available Plugins

The project currently supports two official React plugins for Vite:

- [`@vitejs/plugin-react`](https://github.com/vitejs/vite-plugin-react) – Uses **Babel** for Fast Refresh  
- [`@vitejs/plugin-react-swc`](https://github.com/vitejs/vite-plugin-react-swc) – Uses **SWC** for Fast Refresh  

## Features

- Responsive and modern UI  
- Smooth animations using Framer Motion  
- Sections for About, Projects, Skills, and Contact  
- Fully customizable and easy to extend  

## Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Irfanlam123/irfan-portfolio.git

# Navigate to the project folder
cd irfan-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
